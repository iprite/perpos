## 6. Copy To Avoid

Avoid using these as primary public-facing claims:

```text
Next-Gen Agentic AI ERP
8 Autonomous AI Agents
AI workforce
System of Action
Enterprise AI network
STT
Recall API
Cloud Run worker
```

These may be true internally or useful in technical docs, but they add friction for public landing pages.
