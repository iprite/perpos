# PERPOS Flow — ผู้ช่วย AI ส่วนตัวบน LINE

## Flow คืออะไร

PERPOS Flow คือผู้ช่วย AI ส่วนตัวบน LINE ที่ทำให้งานไฟล์แทบไม่มีแรงเสียดทาน หลักการคือ "โยนไฟล์เข้าไป ยืนยัน รอระบบจัดการ แล้วรับผลลัพธ์กลับมาในแชต" ใช้ได้ทั้งไฟล์เสียงและ PDF จ่ายแบบ token เท่าที่ใช้ ไม่มีค่ารายเดือน และเริ่มต้นใช้งานได้ฟรี

## Flow ทำอะไรได้บ้าง (ฟีเจอร์)

Flow มี 3 ฟีเจอร์หลัก ทุกฟีเจอร์ทำงานบนจังหวะเดียวกัน:

1. **ถอดเสียง + สรุปประชุม** — ส่งไฟล์เสียงหรือวิดีโอประชุม รับ transcript พร้อมรายงานการประชุม (MoM) ที่อ่านต่อได้ทันที
2. **บีบ PDF** — ส่ง PDF เข้า LINE ระบบขอยืนยัน บีบไฟล์ แล้วส่ง PDF ที่เล็กลงกลับให้
3. **Meeting bot** — รองรับ Google Meet, Zoom และ Microsoft Teams ส่งบอทเข้าประชุมเพื่อบันทึกและสรุปให้

## ขั้นตอนการทำงานของ Flow

Flow ทำงาน 4 ขั้นในแชต LINE: **โยนเข้า → ยืนยัน → จัดการ → ส่งไฟล์**

1. โยนไฟล์เสียงหรือ PDF เข้าแชต
2. ระบบขอให้ยืนยันว่าจะทำงานนั้น
3. ระบบประมวลผล
4. ส่งผลลัพธ์ (เช่น transcript, MoM PDF, หรือ PDF ที่เล็กลง) กลับมาในแชต

## เริ่มใช้ Flow อย่างไร

เริ่มใช้งานได้เลยในแชตนี้ ไม่ต้องสมัครหรือเรียนรู้ระบบใหม่ ใช้งานได้ฟรีทันที — แค่ส่งไฟล์เสียงหรือ PDF เข้ามาในแชต ระบบจะขอให้ยืนยันก่อนเริ่มทำงาน แล้วส่งผลลัพธ์กลับมาให้ในแชตเดียวกัน

## ขนาดไฟล์ที่ Flow รองรับ

Flow รองรับไฟล์ขนาดใหญ่ได้สูงสุด 500 MB ทั้ง PDF และไฟล์เสียง

## รายงานการประชุม (MoM) ที่ได้จาก Flow

เมื่อส่งไฟล์เสียงประชุม Flow จะถอดเสียงและสรุปเป็นรายงานการประชุม (Minutes of Meeting) ให้ในรูปแบบ PDF พร้อม transcript ที่อ่านต่อได้ทันที เหมาะกับการเก็บบันทึกและส่งต่อในทีม
