# ความเป็นส่วนตัวและความปลอดภัยของข้อมูล (PERPOS)

## PERPOS เก็บไฟล์ของลูกค้าไหม

ไฟล์งานของคุณถูกใช้เพื่อประมวลผลงานที่สั่งเท่านั้น โดยมีหลักการดังนี้:

- **ไม่เก็บไฟล์เสียงไว้ถาวร** — ไฟล์เสียงไม่ถูกเก็บถาวรในระบบ PERPOS
- **ลบภายใน 48 ชั่วโมง** — ไฟล์สรุปประชุมถูกลบออกจาก server ภายใน 48 ชั่วโมง
- ข้อมูลถูกเก็บบน Google Cloud พร้อมการเข้ารหัส

## PERPOS เอาข้อมูลไปเทรน AI ไหม

ไม่ ไฟล์และเสียงของลูกค้าไม่ถูกนำไปฝึก (เทรน) โมเดล AI ใด ๆ ข้อมูลถูกใช้เพื่อประมวลผลงานที่คุณสั่งและปรับปรุงบริการเท่านั้น

## PERPOS ปลอดภัยและเป็นไปตาม PDPA ไหม

PERPOS ออกแบบให้ตรวจขั้นตอนเรื่องข้อมูลอย่างเข้มงวด (PDPA-aware) เราไม่ขายหรือแชร์ข้อมูลส่วนบุคคลของคุณกับบุคคลที่สามโดยไม่ได้รับความยินยอม ยกเว้นกรณีที่กฎหมายกำหนด

## สิทธิ์ของเจ้าของข้อมูล

คุณยังคงเป็นเจ้าของข้อมูลที่คุณป้อนเข้าสู่ระบบ PERPOS และมีสิทธิ์เข้าถึง แก้ไข หรือลบข้อมูลส่วนบุคคลของคุณได้ตลอดเวลา ผ่านการตั้งค่าบัญชีหรือติดต่อทีมงานที่ hello@perpos.ai

## การชำระเงินและการยกเลิก

Flow คิดค่าบริการแบบ token เติมเท่าที่ใช้ ส่วน Suite เป็น subscription ตามขอบเขตที่ตกลง คุณสามารถยกเลิกได้ตามเงื่อนไขของแต่ละบริการ ข้อมูลนโยบายฉบับเต็มดูได้ที่หน้า Privacy และ Terms บนเว็บไซต์ perpos.ai (อัปเดตล่าสุด: มกราคม 2568)
