# PERPOS Suite — AI ERP สำหรับองค์กร

## Suite คืออะไร

PERPOS Suite คือ AI ERP สำหรับองค์กรที่ต้องการระบบตาม workflow จริง เริ่มจาก module พื้นฐานอย่างบัญชีและ HR แล้วต่อยอดด้วย tailor-made ERP ที่ออกแบบให้เข้ากับกระบวนการของแต่ละองค์กร ใช้งานจริงแล้วในหลายองค์กร Suite ไม่ได้เป็นแค่ catalog module แต่เป็นระบบที่ปรับตามหน้าที่งาน approval, document, operation และรายงานขององค์กรนั้น ๆ

## Module ของ Suite มีอะไรบ้าง

- **บัญชี** — เอกสารบัญชี ภาษี และสมุดรายวัน (Journal) ใน workflow เดียว (Invoice, ภาษี, Journal)
- **บุคคล / HR** — ข้อมูลพนักงาน payroll และ approval ที่ต่อกับโครงสร้างองค์กร (พนักงาน, Payroll, สิทธิ์)
- **ขาย – ซื้อ** — คุมเอกสารขาย-ซื้อ ตั้งแต่ใบเสนอราคา PO ไปจนถึง invoice (Quote, PO, Billing)
- **คลังสินค้า** — ติดตามสินค้า stock movement และต้นทุนที่เกี่ยวกับคลัง (Stock, Movement, ต้นทุน)
- **การเงิน** — จัดการธนาคาร เช็ค payment และกระแสเงินสดขององค์กร (ธนาคาร, เช็ค, Cashflow)
- **Tailor-made ERP** — ออกแบบ workflow เฉพาะองค์กร ใช้งานจริงแล้วในหลายทีม (Production-ready, Custom flow, Integration)

## AI ใน Suite ทำอะไร

AI ใน Suite ไม่ใช่แค่ chatbot แต่ช่วยทำงานในระบบจริง: อ่านเอกสาร สรุปข้อมูล ตรวจงาน และเชื่อมขั้นตอนระหว่างทีม ให้งานซ้ำ ๆ เดินเองได้ ตัวอย่าง automation:

- **OCR เอกสารบัญชี** — อ่านบิลและใบเสร็จ เข้าสู่ระบบอัตโนมัติ
- **สรุปข้อมูลผู้บริหาร** — สรุปยอดและสถานะงานให้อ่านได้เร็ว
- **แจ้งเตือนผ่าน LINE** — ส่ง noti และคำสั่งงานเข้า LINE ของทีม

## ขั้นตอนการเริ่มใช้ Suite (Implementation)

เริ่มต้นเป็นขั้น ไม่ต้องรื้อทั้งระบบ — มี 3 ขั้น:

1. **Discover** — เข้าใจ workflow และข้อมูลที่องค์กรใช้จริง
2. **Build** — ออกแบบ module และ automation ให้เข้ากับทีม
3. **Run** — deploy ใช้งานจริง พร้อมดูแล subscription ต่อเนื่อง

## ขอเดโม Suite อย่างไร

Suite มีบริการนัดเดโมฟรี ทีมงานจะช่วยออกแบบ workflow ให้เข้ากับองค์กรของคุณ ติดต่อขอเดโมได้ที่ hello@perpos.ai
