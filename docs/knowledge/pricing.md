# ราคาและการเติม token ของ PERPOS Flow

## Flow คิดราคาอย่างไร

PERPOS Flow คิดค่าบริการแบบ **token pay-as-you-go** — เติม token เท่าที่ใช้ ไม่มี subscription รายเดือน จ่ายตามการใช้งานจริง เริ่มต้นใช้งานได้ฟรี

## อัตราแลกเปลี่ยน token

- **1 บาท = 100 token** (1 token = 0.01 บาท)
- ตัวอย่างการเติม: **99 บาท = 9,900 token**
- เริ่มต้นเติมได้ที่ 99 บาท
- เติม token ได้ทั้งใน LINE และบนเว็บ
- ผู้ใช้ใหม่ได้ token ทดลองใช้ฟรีเพื่อเริ่มต้น (เริ่มใช้งานได้ทันที ไม่ต้องสมัคร)

## token หมดอายุไหม

- token ที่ **เติมเงิน** มีอายุ **1 ปี (365 วัน)** ต่อก้อนที่เติม
- **เติมเพิ่มก่อนหมดอายุ → ก้อนเก่าทุกก้อนขยายอายุออกไปอีก 1 ปี** (rollover แบบค่ายมือถือ) ดังนั้นถ้าเติมต่อเนื่องภายใน 1 ปี token จะไม่หมดอายุ
- การใช้ token ตัดจากก้อนที่ใกล้หมดอายุก่อน (FIFO)
- token ทดลองใช้ (trial) สำหรับผู้ใช้ใหม่มีอายุสั้นกว่า (ประมาณ 30 วัน)

## ค่าใช้ token ต่อฟีเจอร์ (Flow)

- **ถอดเสียง + สรุปประชุม:** 100 token / นาที (ประมาณ 1 บาท / นาที)
- **บีบ PDF:** 100 token / หน้า (ประมาณ 1 บาท / หน้า)
- **Meeting bot (Meet / Zoom / Teams):** 150 token / นาที (ประมาณ 1.5 บาท / นาที)

## Flow มีค่ารายเดือนไหม

ไม่มี Flow ไม่มีค่าบริการรายเดือนและไม่มี subscription จ่ายเฉพาะ token ที่ใช้จริงเท่านั้น เริ่มใช้งานได้ฟรี เพิ่มเพื่อน @perpos ใน LINE แล้วส่งไฟล์แรกได้ทันที

## ราคาของ Suite

PERPOS Suite (ระบบ ERP องค์กร) คิดราคาแบบ subscription ตามขอบเขตการใช้งานที่ตกลงกัน ไม่ใช่ระบบ token เหมือน Flow สำหรับราคาและขอบเขต แนะนำให้นัดเดโมฟรีกับทีมงานที่ hello@perpos.ai
